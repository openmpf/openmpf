Pipeline Creation

This view allows for the creation of custom actions, tasks, and pipelines. A Custom action is based on an existing algorithm. A custom action will appear in the action selection dropdown once created. Existing and custom actions are used to create a new task. Once a custom task is created it will appear in the task selection dropdown and can be added to a new pipeline along with existing tasks. If an error is received when creating a custom action, task, or pipeline without a complete description, please check the tomcat log.
